# Palavra Viva 3x

Pipeline automático de vídeos bíblicos em 3 idiomas (espanhol, inglês e
português): narração TTS, legendas queimadas e imagens CC0, publicado
diariamente via GitHub Actions.

Todo o texto bíblico vem de traduções em **domínio público** (Reina-Valera
1909, King James Version, Bíblia Livre — ver `LICENCAS.md`). A trilha
ambiente dos vídeos longos é sintetizada proceduralmente pelo próprio
pipeline (`nucleo/musica.py`).
